## Setup

- **.env Requirements**
  - PORT=3000
  - MONGOOSE_URI=mongodb://localhost:27017/flowers

## Running the App

- npm start or nodemon

- **Endpoint**
  - used postman and in terminal HTTP pie

## Test

- run command `npm test`
